# TankTourney

![maintained](https://img.shields.io/badge/Status-maintained-brightgreen.svg)

**TankTourney** is a specialized tournament organization tool designed for the World of Tanks (WoT) competitive scene. Built with Godot 4, it provides high-performance bracket management and real-time synchronization.
